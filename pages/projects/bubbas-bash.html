{% filter markdown %} Bubba's Bash is an annual benefit concert for the Bubba Watson Foundation. I created a responsive site in order to aid in their marketing efforts. The key objectives were to create 3 location specific pages that would allow users to quickly get information about the artists, and find links to purchase tickets and get directions to the event. For this project I used Sketch for the wires / design and a build tool called Docpad that uses a combination of sass (which compiles to css), html and javascript to generate the static pages. Using simple static pages allowed for fast load times and it was easy to iterate the (responsive) design in browser. Visit the site here. {% endfilter %}
